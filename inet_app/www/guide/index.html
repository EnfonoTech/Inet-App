1. Overview

The INET Operations Command Center manages telecom rollout projects from the first purchase order to the final invoice. Every stakeholder — Project Manager, Implementation Manager, Field Team, and PIC — works through the same portal, each seeing only the pages relevant to their role.

Work order lifecycle

PO Upload→ Dispatch→ PO Control→ Rollout Planning→ Rollout Execution→ Work Done→ Invoice

Materials lifecycle (parallel)

Huawei Delivery→ Main Warehouse→ Team Warehouse→ On-site Use→ Return if unused

1.1 User Roles

When you log in the portal automatically shows the correct layout for your role. You only see pages relevant to you.

Project Manager (PM)

Full access: all dashboards, all teams, all projects, financials, approvals, masters.

Implementation Manager (IM)

Their own teams, projects, materials, execution, backend assignment, expense approvals.

Field Team

Today's assigned work, execution forms, team stock, expense submission.

PIC

Milestone billing tracker, invoice pipeline, payment tracking.

1.2 Common Controls

These controls appear on all data pages across the portal.

ControlBehaviour
Search boxFilters the table as you type (300 ms debounce). Searches across the most relevant fields shown in the placeholder text.
Filter dropdownsMulti-select dropdowns for Status, Project, Team, DUID, etc. Click the dropdown, tick one or more options, click away to apply.
Clear buttonResets all active filters. Only appears when at least one filter is set.
Row limit selectorAt the bottom of every table — choose 25 / 50 / 100 / All. "All" returns every matching row.
Export Excel / Download CSVExports the current filtered and sorted view.
Refresh buttonReloads the page data from the server. List pages do not auto-refresh; click Refresh for the latest state.
Manage TableColumn visibility, order, widths, and per-column sort or filter — saved per user across sessions.
